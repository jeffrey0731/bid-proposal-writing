一个适配 Claude Code / Codex 等 Agent 环境的服务方案编写技能，专为采购代理机构参加比选活动设计。核心逻辑：**评分标准决定结构，采购需求决定内容，合同条款决定边界。**

## 核心特性

- **响应驱动编写**：从比选文件的评审标准出发，逐项精准响应，杜绝"自说自话"
- **评审标准映射**：建立评分项与方案章节的完整对照表，确保无遗漏
- **三里程碑管控**：框架确认 → 初步编写 → 扩写完善，每阶段用户确认后再推进
- **分批次交付**：先写方向确认章节，再写核心内容，最后补充完善
- **合同条款覆盖**：付款节点、验收标准、违约责任在方案中逐一回应
- **专业公文规范**：严格遵循中文公文编号体系、排版格式

## 适用场景

- 采购代理机构参加招标人组织的代理机构比选/遴选
- 编写代理服务投标文件
- 响应竞争性磋商/谈判中的代理方案要求

## 工作流程

```
信息收集 → 比选文件深度分析 → 框架结构设计 → 分阶段编写 → 终稿确认
```

### 比选文件分析四维度

1. **评审标准拆解**：逐条拆解评分项，建立响应对照表
2. **采购需求分析**：区分硬性需求、软性需求、隐性需求
3. **合同条款分析**：付款条件、违约责任、验收标准、知识产权
4. **项目属性判定**：采购方式、项目类别、适用法律体系

### 分阶段编写策略

| 阶段 | 内容 | 目的 |
|------|------|------|
| 批次一 | 项目理解 + 人员配置 | 确认方向正确，避免返工 |
| 批次二 | 代理服务方案 + 质量保障 | 核心分值章节，重点打磨 |
| 批次三 | 项目管理 + 增值服务 + 附件 | 补充完善，体现差异化 |

## 通用方案框架

```
前言（项目理解与服务承诺）
第一章  项目理解与需求分析
第二章  代理服务方案（核心）
第三章  项目管理方案
第四章  人员配置方案
第五章  质量保障与风险控制
第六章  增值服务与承诺
附件（资质业绩、人员简历、证明材料）
```

## 编写规范

- **文字格式**：宋体/仿宋小四，1.5倍行距，A4纸张
- **编号体系**：第一章 → 1. → 1.1 → (1) → ①
- **图表规范**：图X-X / 表X-X，300dpi，统一配色
- **篇幅参考**：一般项目 60-120 页，重大项目 120-200 页

## 编写原则

1. **不自说自话**：每段内容都对应评审标准的某一项
2. **不面面俱到**：篇幅与分值权重匹配
3. **不空洞承诺**：写具体措施而非泛泛保证
4. **不忽视合同**：合同关键条款在方案中必须有回应
5. **突出针对性**：结合行业特点和采购人属性
6. **体现专业性**：适当加入行业洞察和增值服务

## 适用法规体系

- 《中华人民共和国招标投标法》及实施条例
- 《中华人民共和国政府采购法》及实施条例
- 《政府采购货物和服务招标投标管理办法》（财政部令第87号）
